Data Cleaning Using SQL

This project focuses on data cleaning using SQL, applied to a dataset of Nashville housing data. It involves loading the raw dataset into a SQL database, identifying and correcting errors, standardizing data, and removing duplicates to ensure data integrity. The project includes three key files: Nashville Housing Data for Data Cleaning.csv (raw data), housing_data_cleaning_queries.sql (SQL queries for cleaning), and load_housing_data.sql (script to load raw data into the database). The objective is to transform raw data into a clean, consistent, and reliable dataset ready for analysis, highlighting the critical role of SQL in automating and streamlining the data cleaning process.

Blog Generation Using Llama 2

This project, "Blog Generation Using Llama 2," leverages the capabilities of the Llama 2 language model to automate the creation of blog content. Using a Python-based application with Streamlit for the user interface and LangChain for prompt management, the project employs the Llama 2 model to generate blog posts tailored to specific styles and topics. Users can input a topic, specify the desired blog style, and set a word limit, after which the model generates a coherent and contextually relevant blog post. This showcases the practical application of advanced natural language processing techniques in content generation, demonstrating how AI can assist in automating creative writing tasks.
